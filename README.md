#  Исследование результатов А/В-теста и поиск инсайтов для приложения
SollmaFin — мобильное приложение для начинающих инвесторов на рынках Латинской Америки. Для данного проекта выдвинута гипотеза о недостаточной финансовой грамотности пользователей, вследствие чего был разработан новый онбординг с подробным описанием рисков различных активов.
Необходимо было изучить исторические данные и провести оценку результатов A/B-эксперимента.

В результате анализа A/B-теста обнаружено, что обновленный онбординг подтвердил все продуктовые гипотезы: конверсия во второй депозит выросла на 60%, а средний чек платежеспособных пользователей увеличился. Однако положительный эффект достигнут за счет оттока начинающих инвесторов с небольшими суммами (конверсия в первый депозит снизилась на 2,11%). Таким образом, новая фича эффективно "фильтрует" аудиторию, повышая лояльность крупных игроков, но требует доработки механик для удержания новичков.

Данные, по которым был выполнен проект, были предоставлены образовательной платформой Яндекс.Практикум в рамках обучения по программе "Аналитик данных. Расширенный курс".

Проект выполнялся с иcпользованием Python и библиотек Pandas, Matplotlib, Seaborn, Scipy, Numpy.
